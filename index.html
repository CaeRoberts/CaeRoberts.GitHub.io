Image Search Application



Overview:

A React based app that uses the Unsplash API to grab images based on user search. The API returns 30 images which can be saved into the localStorage for your viewing pleasure at a later time. The grid-based layout of the image list is mobile friendly and is responive to various screen sizes.


Role:

Personal Project. I was the sole designer and developer. I used React to make the App and tested the website across Google Chrome, Safari, Mozilla Firefox and even Internet Explorer! I used multiple CSS-linked files and semantic-ui for a clean presentation style.
